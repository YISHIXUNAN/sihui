#### 技术栈

```

打包工具：webpack

开发语言：react + ts

组件库：Ant Design

状态管理：mobx

路由：react-router-dom

数据请求：axios

代码规范：eslint + prettier

git相关：.gitignore + commitlint.config.js

```



#### 目录结构


```
├── dist  生产文件
├── src
	├── .sihui   配置封装文件
        └──config    webpack 配置相关
        └──core   axios 等内容的二次封装
    ├── assets   静态文件
    ├── config
    	└── proxy.tsx 请求配置
    	└── routes.tsx 路由配置
    ├── hooks 自定义hook
    ├── pages 页面
    ├── store 状态文件
```
